# 第一篇筆記

這是一篇範例文章。你之後新增文章時，可以複製這個資料夾的格式。

## 建議寫法

每篇文章可以放在自己的資料夾裡：

```text
src/notes/my-new-note/README.md
```

文章最上面的 `---` 區塊叫做 frontmatter，可以設定標題、日期、分類和標籤。

## 下一步

你可以把這篇改成你自己的第一篇文章，例如：

- 今天學到的東西
- 某個作業或專案的紀錄
- 某段程式碼的理解
